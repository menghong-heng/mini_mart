# 67 mini mart

Welcome to **67 mini mart**, a boutique e-commerce web application featuring a sleek, responsive design and robust staff portal. 

## 🚀 Features

- **Customer Storefront:** A premium, fully responsive shopping experience with dark mode support.
- **Staff Portal:** A dedicated, secure backend portal for inventory, order, and user management.
- **Dark Mode:** A seamless, system-synchronized dark mode feature across all customer pages.
- **Authentication:** Role-based access control for both customers and staff.

## 🛠️ Tech Stack

- **Frontend:** React (Vite), Tailwind CSS
- **Backend:** FastAPI, Python 3.12
- **Database:** PostgreSQL 16
- **Deployment:** Docker & Docker Compose

## 📦 Getting Started

### Prerequisites
- Docker and Docker Compose installed on your machine.

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Menghong-Heng/mini_mart.git
   cd mini_mart
   ```

2. Start the application using Docker Compose:
   ```bash
   docker compose up --build -d
   ```

3. Access the application:
   - **Customer Shop:** `http://localhost:5173/`
   - **Staff Portal:** `http://localhost:5173/staff/login`
   - **API Docs:** `http://localhost:8000/docs`
